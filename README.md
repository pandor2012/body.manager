# Панель команд (мобильная веб-версия)

Простое браузерное приложение с кнопками:
- Влево
- Вправо
- Не двигаться
- Лечь на пол
- Присесть

## Локальный запуск

Откройте проект в терминале и выполните:

```bash
python3 -m http.server 8080
```

После этого откройте в браузере:
- на компьютере: `http://localhost:8080`
- на телефоне в той же Wi-Fi сети: `http://<IP_ВАШЕГО_КОМПЬЮТЕРА>:8080`

## Хостинг (рекомендуется)

### Вариант 1: GitHub Pages

Подходит, если код хранится в GitHub.

1. Загрузите проект в репозиторий GitHub.
2. В репозитории откройте `Settings -> Pages`.
3. В `Build and deployment` выберите `Deploy from a branch`.
4. Выберите ветку `main` и папку `/ (root)`.
5. Сохраните. Через 1-2 минуты сайт будет доступен по ссылке вида:
   `https://<username>.github.io/<repo>/`

### Вариант 2: Netlify Drop (самый быстрый без Git)

1. Откройте `https://app.netlify.com/drop`.
2. Перетащите папку проекта в окно браузера.
3. Сразу получите публичную ссылку, которую можно открыть с любого мобильного устройства.
